# Federal_Taxation

Solving US Federal Tax Exercises 

In this repository we use Jupyter Notebooks to solve US Federal Tax exercises using the Python programming language.  The exercises are taken from the `Taxation of Individuals and Business Entities 2025`.  Eventhough, the book was written prior the `One Big Beautiful Act (OBBA)`, we will be adapting the exercises to it whenever possible, using the 2026 US Federal Tax tables.  

For the actual solving and calculation of taxes and tax planning, I use a proprietary software solution written by me and used in my private practice (not included in here). If you would like to learn more about it or want a custom solution for your business feel free to reach out.
